Добро пожаловать!

Используйте приложение для управления продуктами в вашем холодильнике. Сфотографируйте продукты, и ИИ поможет вам с рецептами и заказами.

Инструкция по использованию

1. Сделайте фото полки холодильника

2. Приложение автоматически распознает продукты и создаст список.

3. Сфотографируйте остальные полки холодильника и дождитесь добавления.

4. Если есть продукты, которые камера не находит, их можно добавить вручную.

5. После этого каждый день делайте новое фото холодильника для поддержания актуальной информации.
